L'applicazione è stata sviluppata tramite Visual Studio Code, con l'ausilio delle estensioni Live Server, PHP Server e PHP Intelephense, mentre per il server del database è stato usato il software XAMPP, per l'esecuzione del sito è necessario installare XAMPP, inserire la cartella HtmlFile1 nella directory "\xampp\htdocs\", avviare Visual Studio Code con Live Server e PHP Server attivi, lanciare XAMPP avviando MySQL e Apache, ed infine lanciare index.html cliccando su "PHP Server: Serve Project" (è possibile trovare il comando premendo tasto destro sul documento)
